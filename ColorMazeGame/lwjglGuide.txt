Display Class - The core graphical class of Slick

	create()
		Creates a new window. In order to set size parameters, setDisplayMode
		must be called, first.
		Ex: Display.create();
		
	setDisplayMode(DisplayMode)
		Sets the display mode for a new window. It accepts a DisplayMode object
		as an argument.
		Ex: Display.setDisplayMode(new DisplayMode(<width>, <height>);
		
	update()
		Updates the screen to what has been drawn. Slick automatically uses
		double buffering, so everything drawn before update() is drawn in the
		non-active buffer. This method switches the buffers. NOTE: This method
		automatically updates keyboard and mouse input.
		Ex: Display.update();
		
	destroy()
		Closes the window made by create() and frees the resources it was
		using.
		Ex: Display.destroy();
		
Mouse Class - Gives information on the mouse

	int getX()
		Returns an integer x value of the mouse within the window.
		Ex: int mX = Mouse.getX();
		
	int getY()
		Returns an integer y value of the mouse within the window.
		Ex: int mY = Mouse.getY();
		
	boolean isButtonDown(int button)
		Returns a true/false value for the given mouse button, represented by
		the "button" integer. Left button = 0, right button = 1.
		Ex: boolean leftDown = Mouse.isButtonDown(0);
		
Keyboard Class - Gives information on the keyboard

	boolean isKeyDown(int key)
		Returns a true/false value for the given keyboard key, represented by
		the "key" integer. Values are Keyboard.KEY_<value>.
		Ex: boolean rightKeyDown = Keyboard.isKeyDown(Keyboard.KEY_RIGHT);
